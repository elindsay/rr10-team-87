instalover

Shutdown

instalover is no more. It was a fun experiment, part of a good competition, and a great team-building exercise, and now it is time to move on.

We, the instalover team, have had to face the reality: building a Web site may take a weekend, but running it takes so much more. instalover is an idea that could have taken off with more user experience, promotion, and feature work, but we just didn't have it in us.

Moving On

For you nerds out there we have opened the code. There you will find a well-tested Rails app that can be used to build toward the future.

In addition our competitor, How About We, is going full-swing at tackling the problem of getting people outside.

Thanks!
